Here is a clean and straightforward Python implementation to generate the Fibonacci sequence.

I have included two approaches: a simple iterative loop (which is efficient and great for printing the sequence up to n terms) and a recursive function (which is often asked in interviews but is less efficient for large numbers).

Method 1: Iterative Approach (Recommended)

This approach uses a loop to generate the sequence, making it highly efficient with a time complexity of O(n).

Python
def print_fibonacci(n):
    """
    Prints the first n numbers in the Fibonacci sequence.
    """
    if n <= 0:
        print("Please enter a positive integer.")
        return
    
    # Initialize the first two terms
    a, b = 0, 1
    
    print(f"First {n} Fibonacci numbers:")
    for _ in range(n):
        print(a, end=" ")
        # Update values: next number is the sum of previous two
        a, b = b, a + b
    print()  # For a newline at the end

# Example usage: Print the first 10 terms
print_fibonacci(10)

Method 2: Recursive Approach

If you are practicing for CS exams or interviews, you might be asked to find the n-th Fibonacci number using recursion.

Python
def fibonacci_recursive(n):
    """
    Returns the n-th Fibonacci number.
    Index starts at 0 (0th number = 0, 1st number = 1).
    """
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci_recursive(n - 1) + fibonacci_recursive(n - 2)

# Printing the first 10 terms using recursion
terms = 10
print(f"First {terms} Fibonacci numbers (Recursive):")
for i in range(terms):
    print(fibonacci_recursive(i), end=" ")
print()


⚠️ Note on Recursion: While elegant, the simple recursive approach has an exponential time complexity of O(2
n
) because it recalculates the same values multiple times. For larger values of n, you'd want to use dynamic programming (memoization) to optimize it!
